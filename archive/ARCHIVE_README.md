# Archive Directory

Acest director conține fișiere arhivate care nu mai sunt necesare pentru funcționarea normală a platformei.

## Structură:

- **`old_docs/`** - Documentație veche și ghiduri depășite
- **`old_reports/`** - Rapoarte vechi de verificare și fix-uri
- **`old_scripts/`** - Script-uri Python vechi și script-uri de test
- **`validation_reports/`** - Rapoarte JSON de validare agenți și reindexare Qdrant

## Notă:

Aceste fișiere sunt păstrate pentru referință istorică, dar nu sunt necesare pentru funcționarea platformei.
